gh-proxy

GitHub 资源代理

支持 release、archive、blob/raw、gist，兼容完整 GitHub URL 和短路径 /owner/repo/...。

支持完整链接和短路径输入；复制按钮会生成完整代理地址。

短路径

owner/repo/releases/download/v1.0.0/file.zip owner/repo/archive/master.zip owner/repo/blob/master/README.md

完整链接

https://github.com/owner/repo/releases/download/v1.0.0/file.zip https://github.com/owner/repo/archive/master.zip

支持内容

  • release 文件
  • archive 源码包
  • blob/raw 文件
  • gist 原始文件

部署

  • Cloudflare Workers
  • Python 3.11+
  • Gunicorn / 自定义反代
项目仓库：lgpay/gh-proxy
