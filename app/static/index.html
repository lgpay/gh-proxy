gh-proxy · GitHub 资源代理 / 加速

更短的 GitHub 代理路径，
更干净的自部署体验。

支持 release、archive、blob/raw、gist，以及简短路径格式： /owner/repo/...。 适合自部署到 Cloudflare Workers 或 Python 3.11+ 服务端。

支持两种输入：完整 GitHub URL，或简短仓库路径。示例见下方。

示例：短路径

owner/repo/releases/download/v1.0.0/file.zip owner/repo/archive/master.zip owner/repo/blob/master/README.md

示例：完整链接

https://github.com/owner/repo/releases/download/v1.0.0/file.zip https://github.com/owner/repo/archive/master.zip https://gist.githubusercontent.com/user/id/raw/file.txt

支持内容

  • release 文件
  • archive 源码包
  • blob/raw 文件
  • gist 原始文件
  • 部分 git clone / info refs

部署形态

  • Cloudflare Workers
  • Python 3.11+ + Gunicorn
  • 自定义域名反代
  • 适合私有或小范围公共服务
